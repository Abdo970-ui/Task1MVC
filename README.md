# Task1MVC – Doctor Appointment System

📌 **Task1MVC** هو أول مشروع ليّ باستخدام **ASP.NET Core MVC**،  
وهو عبارة عن **نظام بسيط لحجز مواعيد الدكاترة**، الهدف منه تطبيق المفاهيم الأساسية للـ MVC و Entity Framework بشكل عملي.

---

## 🚀 Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- LINQ
- SQL Server
- HTML, CSS


---

## ✨ Features
- ✅ عرض قائمة الدكاترة
- ✅ البحث عن دكتور بالاسم
- ✅ تصفية الدكاترة حسب التخصص
- ✅ Pagination لعرض البيانات بشكل منظم
- ✅ إنشاء مواعيد (Appointments)
- ✅ ربط الموعد بالدكتور
- ✅ تطبيق MVC Architecture (Model – View – Controller)

---

## 🧠 What I Learned
من خلال المشروع ده تعلمت:
- استخدام **DbContext** والتعامل مع قواعد البيانات
- عمل Relationships بين الـ Models
- الفرق بين **HTTP GET & POST**
- تمرير البيانات للـ Views باستخدام **ViewBag**
- استخدام **LINQ** لفلترة البيانات
- تنظيم الكود داخل Controllers و Views
